# PPT 助手 - AI 驱动模式设计

## 核心架构

### 1. 对话历史管理
- 每个用户维护独立的对话历史
- 保存最近 20 轮对话
- 支持上下文理解

### 2. AI 意图识别
用 Kimi API 分析用户消息，识别意图：
- `generate` - 生成新 PPT
- `modify` - 修改现有内容
- `adjust` - 调整样式/格式
- `question` - 询问/讨论
- `confirm` - 确认/继续
- `cancel` - 取消/重置

### 3. 灵活响应
不固定流程，根据用户意图动态响应：
- 可以随时修改之前的内容
- 可以跳过某些步骤
- 可以反复调整直到满意

### 4. 状态管理
简化状态，只记录：
- 当前主题
- 当前大纲（如有）
- 当前详细内容（如有）
- 对话历史

## 示例交互

```
用户：帮我做个PPT，关于成都写字楼市场
AI：好的，我来为您制作。请问您希望侧重哪些方面？比如市场现状、投资建议、区域分析等？

用户：重点讲投资建议和区域分析
AI：明白。我先为您生成一个大纲：
    1. 封面：成都写字楼投资分析
    2. 市场概况
    3. 重点区域分析（高新、天府新区）
    4. 投资建议
    5. 风险提示
    6. 结束页
    
    您看这样可以吗？需要调整吗？

用户：第3页改成"核心商务区对比"
AI：好的，已修改：
    3. 核心商务区对比（锦江、高新、天府新区）
    
    还有其他需要调整的吗？

用户：可以了，生成详细内容吧
AI：正在生成详细内容...
    [生成后展示预览]
    您看内容如何？需要补充或修改吗？

用户：第4页投资建议部分，加上2024年最新数据
AI：好的，我来添加2024年最新数据...
    [更新后展示]
    已更新，还有其他需要调整的吗？

用户：没问题了，生成PPT文件
AI：正在生成PPT文件...
    ✅ PPT已生成！
    
    如需修改，随时告诉我！
```

## 技术实现

### 系统提示词
```
你是专业的PPT制作助手，擅长：
1. 根据用户需求设计PPT结构和内容
2. 生成专业、有深度的商业PPT
3. 灵活调整，满足用户的个性化需求
4. 保持对话流畅，像专业顾问一样交流

工作流程：
1. 了解用户需求和偏好
2. 设计大纲并征求确认
3. 生成详细内容并展示预览
4. 根据反馈调整优化
5. 生成最终PPT文件

原则：
- 主动询问，确保理解用户需求
- 提供选项，让用户做决策
- 随时接受修改，不限制修改次数
- 保持专业但友好的语气
```

### 函数调用
Kimi API 支持 function calling，可以定义：
- `generate_outline` - 生成大纲
- `generate_content` - 生成详细内容
- `create_ppt` - 生成PPT文件
- `modify_section` - 修改特定部分

让 AI 自主决定调用哪个函数。
